---
title: ボイス選択
description: ボイス名 (-v) と国・言語コード (-L) によるボイス選択
---

`jmactts` は 2 通りの方法でボイスを指定できます。`-v` (ボイス名) と `-L` (国・言語コード) で、両方指定された場合は **`-v` が優先**されます。

## 国・言語コードで自動選択 (`-L`)

`-L` だけ与えると、その言語/国のプライマリボイスが自動で選ばれます。ボイス名を覚えていなくても言語を切り替えられます。

```bash
jmactts -L ja_JP こんにちは     # 完全ロケール → Kyoko
jmactts -L ja こんにちは         # 言語コード → Kyoko
jmactts -L JP こんにちは         # 国コード → Kyoko
jmactts -L en "Hello"           # 英語 → Samantha
jmactts -L en_GB "Cheerio"      # イギリス英語 → Daniel
```

### 照合ロジック

`-L` の値は次の順で照合されます。

1. **完全ロケール** (`ja_JP` のようにアンダースコアを含む) → 完全一致
2. **言語コード** (`ja` / `en` 等) → `xx_*` の全マッチ
3. **国コード** (`JP` / `US` 等) → `*_YY` の全マッチ

### プライマリボイスの選び方

マッチしたボイスが複数ある場合、**名前にカッコ `(` を含まないもの**が優先されます。

macOS では各言語のプライマリボイス (`Kyoko` / `Samantha` / `Daniel` 等) はカッコなしで命名され、補助ボイス (`Eddy (日本語（日本）)` 等) はカッコ付きで命名されているためです。

## ボイス名で直接指定 (`-v`)

`say -v ?` で確認できるボイス名をそのまま指定します。

```bash
jmactts -v Kyoko こんにちは
jmactts -v Samantha "Hello world"
jmactts -v Otoya こんばんは
```

## ボイス一覧

`-l` で利用可能なボイスを一覧表示します。`-L` と組み合わせると絞り込めます。

```bash
jmactts -l                # 全ボイス (サンプルテキスト付き)
jmactts -l -L ja          # 日本語ボイスのみ
jmactts -l -L en_GB       # イギリス英語のみ
```

## 話速の調整 (`-r`)

`-r` で話速 (words per minute) を変更できます。標準値はボイスごとに異なります。

```bash
jmactts -L ja -r 250 早口で読み上げる
jmactts -L ja -r 120 ゆっくり読み上げる
```
