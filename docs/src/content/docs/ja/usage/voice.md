---
title: ボイス選択
description: ボイス名 (-v) と言語/国コード (-L) によるボイス選択
---

`jmactts` は 2 通りの方法でボイスを指定できます。

## ボイス名で指定 (`-v`)

`say -v ?` で確認できる正確なボイス名を指定します。

```bash
jmactts -v Kyoko こんにちは
jmactts -v Samantha Hello world
```

利用可能なボイス一覧は `-l` で確認できます。

```bash
jmactts -l                # 全ボイス (sample 付き)
jmactts -l -L ja          # 日本語ボイスのみ
```

## 言語/国コードで自動選択 (`-L`)

`-v` を指定せず `-L` のみ与えると、その言語/国のプライマリボイスが自動選択されます。

```bash
jmactts -L ja_JP こんにちは     # 完全ロケール
jmactts -L ja こんにちは         # 言語コード
jmactts -L JP こんにちは         # 国コード
jmactts -L en Hello              # en_* のいずれか
```

### 照合ロジック

`-L` の値は以下の順で照合されます。

1. `ja_JP` 形式 (アンダースコア含む) → ロケール完全一致
2. 言語コード (`ja`, `en` 等) → `xx_*` の全マッチ
3. それでもマッチしない場合は国コード (`JP`, `US` 等) として `*_YY` を探索

### プライマリボイスの選び方

複数マッチした場合、**ボイス名にカッコ `(` を含まないもの**を優先します。

macOS の慣習で、各言語のプライマリボイス (`Kyoko` / `Samantha` / `Daniel` 等) はカッコなしの命名、補助ボイス (`Eddy (日本語（日本）)` 等) はカッコ付きの命名になっているためです。

## 速度の調整 (`-r`)

`-r` で話速 (words per minute) を変更できます。

```bash
jmactts -L ja -r 250 早口で読み上げ
jmactts -L ja -r 120 ゆっくり読み上げ
```
